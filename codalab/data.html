Data & Starting Kit

During the challenge, each phase (i.e. Feedback Phase or Final Phase) has 5 meta-datasets, each from a different image "domains". Each meta-dataset is used to generate few-shot learning tasks via an API from the Meta-dataset repo. For more details, please refer to the provided

Starting Kit

This starting kit provides a walkthrough to making a valid submission with an example meta-dataset: Omniglot.

For a quick submission, you can directly download this mysubmission.zip file and submit it by clicking on the button "Upload a Submission". Note that this file can also be generated by going through the tutorial notobook in the starting kit.

All images in the challenge are of shape [128,128,3].

Feedback data: The data is kept hidden but you can get quick feedback on the leaderboard.

Final data: This phase will start at the very end of the challenge. The data is kept hidden.
